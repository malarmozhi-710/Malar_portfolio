## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/malarmozhi-710/Malar_portfolio.git
```

Navigate to the project folder:

```bash
cd Malar_portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
